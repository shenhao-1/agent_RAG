# Agent RAG - 智能客服系统

一个基于 ReAct 框架的智能客服系统，结合检索增强生成（RAG）技术，使用 Streamlit 提供交互式用户界面。

## 📋 项目简介

**Agent RAG** 是一个集成了大语言模型、智能代理（Agent）和检索增强生成技术的综合解决方案。该系统支持流式响应、多工具集成和智能决策，可用于构建高效的智能客服、知识库查询等应用。

### 核心特性

- 🤖 **ReAct 智能代理**：基于 ReAct 框架的推理和行动循环
- 📚 **RAG 能力**：支持检索增强生成，提升答案准确性
- 🔄 **流式输出**：实时响应流，提升用户体验
- 🛠️ **多工具集成**：内置多个实用工具（天气查询、用户信息、数据分析等）
- 💾 **向量数据库**：基于 Chroma 的高效向量存储和检索
- 🎛️ **可配置系统**：灵活的提示词和配置管理
- 📊 **中间件支持**：工具监控、日志记录、提示词切换等
